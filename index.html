Wat is Bootstrap?

Bootstrap is een responsive en mobile first library. Het is een veelzijdig systeem wat gebruikt kan worden in zowel de prototype fase als in het eind product.

Bootstrap maakt gebruik van HTML, CSS en jQuery. Alle basis mogelijkheden van Bootstrap staan beschreven in hun uitgebreidde documentatie. Deze beschrijvingen bestaan direct uit code met uitleg, hierdoor is alleen knip en plak werk nodig.

Uitbreidingen aan de code zijn makkelijk toe te voegen: zelf schrijven of andere plugins zoals Chart.JS of SASS.

Wie gebruikt er Bootstrap?

Wereldwijd gebruiken heel veel mensen Bootstrap: programmeurs, webdevelopers, webdesigners, UI/UX designers, prototypers en werknemers bij grote bedrijven (Twitter, Spotify, Snapchat etc).

Waarom moet ik Bootstrap gebruiken?

Allereerst moet niks! Maar basiskennis van Bootstrap kan handig zijn. Zoals je hebt gelezen werken ook grote bedrijven met Bootstrap, hier moet je als designer rekening mee kunnen houden in je werk. Bijvoorbeeld met het 12 kolommen systeem en kolommen onder elkaar schuiven bij mobiel.

Daarnaast is het een snelle manier voor je eigen (online) prototypes. In een uur kan je een hele basis website in elkaar zetten om je design te testen bij de gebruiker. Je kan het op deze manier makkelijke testen zonder dat er veel tijd besteed wordt aan een definitief ontwerp en details.

Waarom wordt Bootstrap gebruikt?

Bootstrap werkt snel en eenvoudig. De meeste code kun je knippen en plakken uit de documentatie, waardoor tijd wordt bespaard. Dat veel code te koppieren is van orginele documentatie, zorgt er ook voor dat de code consistent is, makkelijk vindbaar en wereldwijd iedereen dezelfde kennis heeft.

De code die je knipt en plakt is mobile first geschreven, waardoor je alleen kleine aanpassingen hoeft te doen voor desktop of tablet. Doordat er maar kleine aanpassingen nodig zijn om het te customizen, is het makkelijk te integreren met andere talen zoals SASS of Chart.JS
